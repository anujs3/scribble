# Scribble
## An Android App Developed in Android Studio

#### Short Description: This Android app can be used to draw quick sketches. It's like a portable whiteboard.

**Features**:
- Draw on the canvas.
- Change pen color.
- Change background color.
- Clear the canvas.
- Undo last pen movement.

**Use Cases**:
- Doodling
- Tic Tac Toe
- Hangman
